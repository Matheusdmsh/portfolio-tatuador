# Portfolio de Tatuador

Site de apresentação de portfólio de tatuador com estilos como Realismo, BlackWork, Fineline, Pontilhismo e outros.

## Descrição

Este projeto é um portfolio online para um tatuador, exibindo galerias de trabalhos por estilo. É construído com HTML, CSS e JavaScript e usa imagens locais para demonstrar diferentes técnicas de tatuagem.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Assets locais (`imagens/`)

## Funcionalidades

- Galeria de imagens organizada por estilo
- Layout responsivo para dispositivos móveis
- Navegação simples e visual atraente

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```
2. Abra o arquivo `index.html` no navegador.

> Dica: para testar localmente de forma mais real, use o GitHub Pages ou um servidor estático como Live Server no VS Code.

## Estrutura do projeto

- `index.html` - página principal
- `style.css` - estilos visuais
- `script.js` - interações e comportamento
- `imagens/` - fotos do portfólio
